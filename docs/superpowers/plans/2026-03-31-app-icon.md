# App Icon Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the placeholder launcher icon with a geometric zodiac wheel icon using Android adaptive icon format.

**Architecture:** Create a vector drawable foreground (the wheel design) and a solid color background, combined via an adaptive icon XML. The wheel geometry is scaled to fit within the 66% adaptive icon safe zone. No legacy PNG generation -- the vector approach covers API 21+ via AppCompat.

**Tech Stack:** Android Vector Drawables (XML), Adaptive Icons (API 26+)

**Spec:** `docs/superpowers/specs/2026-03-31-app-icon-design.md`
**Preview:** `docs/icon-preview.html`

### Task 3: Create the adaptive icon XMLs

Replace the existing placeholder adaptive icon with a proper one in `mipmap-anydpi-v26`, and create a round variant.

**Files:**
- Create: `app/src/main/res/mipmap-anydpi-v26/ic_launcher.xml`
- Create: `app/src/main/res/mipmap-anydpi-v26/ic_launcher_round.xml`
- Delete: `app/src/main/res/mipmap-mdpi/ic_launcher.xml`

- [ ] **Step 1: Create the mipmap-anydpi-v26 directory**

```bash
mkdir -p app/src/main/res/mipmap-anydpi-v26
```

- [ ] **Step 2: Create ic_launcher.xml**

```xml
<?xml version="1.0" encoding="utf-8"?>
<adaptive-icon xmlns:android="http://schemas.android.com/apk/res/android">
    <background android:drawable="@color/ic_launcher_background"/>
    <foreground android:drawable="@drawable/ic_launcher_foreground"/>
</adaptive-icon>
```

- [ ] **Step 3: Create ic_launcher_round.xml (same content)**

```xml
<?xml version="1.0" encoding="utf-8"?>
<adaptive-icon xmlns:android="http://schemas.android.com/apk/res/android">
    <background android:drawable="@color/ic_launcher_background"/>
    <foreground android:drawable="@drawable/ic_launcher_foreground"/>
</adaptive-icon>
```

- [ ] **Step 4: Delete the old placeholder icon**

```bash
rm app/src/main/res/mipmap-mdpi/ic_launcher.xml
```

If the `mipmap-mdpi` directory is now empty, remove it:

```bash
rmdir app/src/main/res/mipmap-mdpi 2>/dev/null || true
```

- [ ] **Step 5: Update AndroidManifest.xml if needed**

Check `AndroidManifest.xml` for the icon reference. It should already have:
```xml
android:icon="@mipmap/ic_launcher"
```

If there's no `android:roundIcon` attribute, add it:
```xml
android:roundIcon="@mipmap/ic_launcher_round"
```

- [ ] **Step 6: Commit**

```bash
git add app/src/main/res/mipmap-anydpi-v26/
git add app/src/main/AndroidManifest.xml
git rm app/src/main/res/mipmap-mdpi/ic_launcher.xml 2>/dev/null || true
git commit -m "feat: replace placeholder icon with adaptive zodiac wheel icon"
```
